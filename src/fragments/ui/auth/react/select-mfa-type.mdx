**Usage**

```jsx
// Required in order to not have the default message of "Less than two mfa types available"
const MFATypeOptions = {
  SMS: true,
  Optional: true,
  TOTP: true,
};

<AmplifySelectMfaType MFATypes={MFATypeOptions}></AmplifySelectMfaType>
```
