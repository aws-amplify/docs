Expand **Gradle Scripts**, open **build.gradle (Module :app)**. You will already have configured Amplify by following the steps in the [Project Setup walkthrough](/gen1/[platform]/prev/start/project-setup/create-application/).

Add API by adding these libraries into the `dependencies` block:
```groovy
dependencies {
    implementation 'com.amplifyframework:aws-api:ANDROID_V1_VERSION'
    implementation 'com.amplifyframework:aws-auth-cognito:ANDROID_V1_VERSION'

}
```

Click **Sync Now**.
