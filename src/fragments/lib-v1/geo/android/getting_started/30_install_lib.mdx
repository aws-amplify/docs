Add the following dependencies to your **build.gradle (Module :app)** file and click "Sync Now" when prompted:

```groovy
dependencies {
    implementation 'com.amplifyframework:aws-auth-cognito:ANDROID_V1_VERSION'
    implementation 'com.amplifyframework:aws-geo-location:ANDROID_V1_GEO_VERSION'
}
```

<Callout>

**Note:** The Geo plugin has a dependency on Cognito Auth.

</Callout>
