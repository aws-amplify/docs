Add the following dependency to your **build.gradle (Module :app)** file and click "Sync Now" when prompted:

```groovy
dependencies {
    implementation 'com.amplifyframework:aws-auth-cognito:ANDROID_V1_VERSION'
    implementation 'com.amplifyframework:aws-geo-location:ANDROID_V1_GEO_VERSION'

    // Add this dependency to integrate MapLibre into your app
    implementation 'com.amplifyframework:maplibre-adapter:ANDROID_V1_GEO_VERSION'
}
```

<Callout>

**Note:** the minimum API level required for the UI components is 21.

</Callout>
