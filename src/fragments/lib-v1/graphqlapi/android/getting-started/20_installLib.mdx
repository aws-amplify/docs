Expand **Gradle Scripts**, open **build.gradle (Module :app)**. You will already have configured Amplify by following the steps in the [Project Setup walkthrough](/[platform]/prev/start/project-setup/create-application/).

Add these libraries into the `dependencies` block:
```groovy
dependencies {
    implementation 'com.amplifyframework:core:ANDROID_V1_VERSION'
    implementation 'com.amplifyframework:aws-api:ANDROID_V1_VERSION'
}
```

Click **Sync Now**.
