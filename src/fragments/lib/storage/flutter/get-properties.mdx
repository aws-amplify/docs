You can retrieve file properties and metadata from S3 storage without downloading the file using `Amplify.Storage.getProperties`.

```dart
Future<void> retrieveProperties() async {
  try {
    final result = await Amplify.Storage.getProperties(
      key: 'example.txt',
      options: const S3GetPropertiesOptions(
        accessLevel: StorageAccessLevel.private,
      ),
    ).result;

    safePrint('file size: ${result.storageItem.size}');
  } on StorageException catch (e) {
    safePrint(e.message);
  }
}
```
