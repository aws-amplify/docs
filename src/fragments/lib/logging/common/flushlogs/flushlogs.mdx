## Change automatic log flush interval
To change the time interval when logs are automatically flushed and sent to CloudWatch.

Below is an example of setting the time interval to 120 seconds:

import iosflushInterval from '/src/fragments/lib/logging/ios/flushlogs/flush_interval.mdx';
import androidflushInterval from '/src/fragments/lib/logging/android/flushlogs/flush_interval.mdx';
import commonflushInterval from '/src/fragments/lib/logging/common/flushlogs/flush_interval.mdx';

<BlockSwitcher>
    <Block name="With Configuration File">
        <Fragments fragments={{all: commonflushInterval}}/>
    </Block>
    <Block name="With Code">
        <Fragments
            fragments={{
                ios: iosflushInterval,
                android: androidflushInterval
            }}/>
    </Block>
</BlockSwitcher>
