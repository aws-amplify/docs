## Force flush logs
If there are exceptional use cases where logs need to be sent immediately.  You can use the plugin to force an automatic log flush.

import iosFlushLogs from '/src/fragments/lib/logging/ios/advanced_plugin_behavior/10_flush_logs.mdx';
import androidFlushLogs from '/src/fragments/lib/logging/android/advanced_plugin_behavior/10_flush_logs.mdx';

<Fragments
  fragments={{
    ios: iosFlushLogs,
    android: androidFlushLogs
  }}
/>
