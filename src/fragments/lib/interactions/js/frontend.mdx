<Callout>

Make sure that the `@aws-amplify/interactions` package has the same version number as the `aws-amplify` package in your `package.json` file. 

</Callout>

Add the interactions package with npm:

```sh
npm install --save @aws-amplify/interactions aws-amplify

```

or with yarn:

```sh

yarn add @aws-amplify/interactions aws-amplify

```
