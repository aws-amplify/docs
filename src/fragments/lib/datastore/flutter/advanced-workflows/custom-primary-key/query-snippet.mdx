Query by primary key value:

```dart
final book = (
  await Amplify.DataStore.query(
    Book.classType,
    where: Book.ISBN.eq('12345'),
  ),
)[0];
```

Query by model identifier of the model:

You should always query by model identifier if the model has a composite primary key.

```dart
// Create a query predicate with a new instance of the model identifier
final book = (
  await Amplify.DataStore.query(
    Book.classType,
    where: Book.MODEL_IDENTIFIER.eq(
      // BookModelIdentifier is a codegen generated class that is exported
      // from ModelProvider.dart
      BookModelIdentifier(
        isbn: '12345',
      ),
    ),
  ),
)[0];

// Create a query predicate with model identifier of an existing instance
// of the Book model
final reQueriedBook = (
  await Amplify.DataStore.query(
    Book.classType,
    where: Book.MODEL_IDENTIFIER.eq(
      book.modelIdentifier,
    ),
  ),
)[0];
```
