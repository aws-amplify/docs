Amplify Geo provides APIs and map UI components for mobile app development such that you can add maps to your app in just a few lines of code. Amplify Geo APIs are powered by [Amazon Location Service](https://aws.amazon.com/location/) and the map UI components from [MapLibre](https://maplibre.org/) are already integrated with the Geo APIs. You can quickly get started using [Amplify CLI](/[platform]/build-a-backend/more-features/geo/configure-maps/) to provision your map resources.

Follow this guide to get started with Amplify Geo through the Amplify CLI.

**Note:**

- If you want to use existing Amazon Location Service resources [follow this guide](/[platform]/build-a-backend/more-features/geo/existing-resources/) instead.
- If you want to use Amazon Location Service APIs not directly supported by Geo, use the [escape hatch](/[platform]/build-a-backend/more-features/geo/amazon-location-sdk/) to access the Amazon Location Service SDK.
