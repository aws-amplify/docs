Amplify uses SQLite via the drift package. This requires a custom build of SQLite when targeting web. You will need to download the a sqlite3.wasm file from [releases](https://github.com/simolus3/sqlite3.dart/releases). Find the latest `sqlite3 #.#.#`, expand Assets and download the sqlite3.wasm

For more context info, see https://drift.simonbinder.eu/web/#drift-wasm
