In your Flutter project directory, open **pubspec.yaml**.

> You will already have configured Amplify by following the steps in the project setup.

Add Analytics by adding these libraries into your dependencies block:

<BlockSwitcher>

<Block name="Stable (Mobile)">

```yaml
environment:
  sdk: ">=2.15.0 <3.0.0"

dependencies:

  # Should already be added during Project Setup walkthrough
  amplify_flutter: ^0.6.0

  # Add these lines in `dependencies` if you have not added it earlier during the Project Setup
  amplify_auth_cognito: ^0.6.0
  amplify_analytics_pinpoint: ^0.6.0
```

</Block>

<Block name = "Developer Preview (Mobile, Web & Desktop)">

```yaml
environment:
  sdk: ">=2.17.0 <3.0.0"

dependencies:
  amplify_analytics_pinpoint: ^1.0.0-next.1
  amplify_auth_cognito: ^1.0.0-next.1
  amplify_flutter: ^1.0.0-next.1
  flutter:
    sdk: flutter

```

</Block>

</BlockSwitcher>
