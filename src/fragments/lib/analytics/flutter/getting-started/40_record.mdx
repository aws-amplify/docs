## Record events

To record an event, create an `AnalyticsEvent` and call `Amplify.Analytics.recordEvent()`:

```dart
Future<void> trackEventsWithProperties() async {
  final event = AnalyticsEvent('test');

  event.customProperties.addBoolProperty('boolKey', true);
  event.customProperties.addDoubleProperty('doubleKey', 10.0);
  event.customProperties.addIntProperty('intKey', 10);
  event.customProperties.addStringProperty('stringKey', 'stringValue');

  await Amplify.Analytics.recordEvent(event: event);
}
```
