## Disable Analytics

Analytics is enabled by default when you configure it in your app. To disable Analytics in your app use the `disable` function:

import android0 from '/src/fragments/lib-v1/analytics/android/enable-disable/disable.mdx';

<Fragments fragments={{ android: android0 }} />

import ios0 from '/src/fragments/lib-v1/analytics/ios/enable-disable/disable.mdx';

<Fragments fragments={{ ios: ios0 }} />

## Enable Analytics

To enable analytics you can use the `enable` function in your app:

import android1 from '/src/fragments/lib-v1/analytics/android/enable-disable/enable.mdx';

<Fragments fragments={{ android: android1 }} />

import ios1 from '/src/fragments/lib-v1/analytics/ios/enable-disable/enable.mdx';

<Fragments fragments={{ ios: ios1 }} />
